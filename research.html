Research

My research focuses on Bengali NLP and practical ML evaluation. I work with transformer-based models for classification (fake reviews/news, sentiment), study dataset balancing and error patterns, and build deployable pipelines.

1) Bengali Fake Review Detection

Explored transformer-based Bengali review classification by training and comparing pretrained models. Emphasis was placed on dataset balancing, class-wise behavior, and error analysis to understand where models fail and why.

BanglaBERT XLM-R Evaluation Error Analysis
GitHub Repo

2) Bengali Fake News Detection

Built and evaluated Bengali misinformation classification pipelines using transformer models. Compared training settings and preprocessing choices to improve generalization and reliability across unseen news-style text.

NLP Transformers Metrics Pipeline
GitHub Repo

3) Sentiment Analysis of Bengali Movie Reviews

Fine-tuned transformer models for Bengali sentiment classification with efficient training methods. Applied PEFT/LoRA-style tuning and tracked performance across different settings to ensure stable, reproducible results.

Transformers LoRA / PEFT Evaluation Bengali NLP
